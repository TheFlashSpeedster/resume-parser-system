Smart Resume Parsing with Hybrid NLP + Rule-Based Extraction

Upload a resume and instantly extract structured fields such as name, email, phone, skills, education, experience, projects, and certifications. The system follows the modular flow: extract → clean → parse → store → display.

Upload Resume

User Features

  • Upload one resume without login
  • Supports PDF, DOCX, and DOC with fallback handling
  • View parsed structured results
  • Download parsed JSON output
  • Readable error/status feedback

Admin Features

  • Session-based secure admin login
  • View all uploaded resumes and details
  • Search, filter, sort, export, and delete records
  • Manual edits with updated timestamp and actor
  • Duplicate detection via email/phone matching

Pipeline Alignment

  • 1.0 Upload Resume
  • 2.0 Extract Text
  • 3.0 Parse Resume
  • 4.0 Store Data
  • 5.0 Display Results
  • 6.0 Admin Management

Supported Extraction Fields

Full Name Emails Phones Location Skills Education Experience Projects Certifications Confidence Scores

This project is designed for academic use and can be extended with stronger NLP models and ATS scoring.
